# Ezenwugo Wilson — Portfolio

A premium, black-and-gold portfolio site built with React, Vite, and Tailwind CSS.

## Getting Started

npm install
npm run dev

Visit http://localhost:5173 to view it locally.

## Build for Production

npm run build
npm run preview

## Deploy to Vercel

1. Push this repo to GitHub (already done).
2. Go to vercel.com/new and import this repository.
3. Vercel auto-detects Vite — leave the defaults:
   - Build Command: npm run build
   - Output Directory: dist
4. Click Deploy.

## Before You Launch

- Add your real photo and project screenshots to src/assets/
- Swap the placeholder Live Demo / GitHub links in PROJECTS (in src/App.jsx)
- Update the social links in the Footer component
